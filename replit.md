# OpUtilityBot

## Overview
A comprehensive Discord utility bot providing various tools and features for server management and user interaction.

## Architecture
- **Backend**: Node.js with Express and Discord.js
- **Frontend**: React with Tailwind CSS and shadcn/ui
- **Database**: PostgreSQL with Drizzle ORM
- **Storage**: Centralized storage layer in `server/storage.ts`

## Key Files
- `server/bot.ts`: Main Discord bot logic
- `server/routes.ts`: API endpoints for the dashboard
- `shared/schema.ts`: Database schema definitions
- `client/src/App.tsx`: Frontend application entry point
